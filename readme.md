# 🏋️ FitTrack

![Java](https://img.shields.io/badge/Java-21-orange)
![NetBeans](https://img.shields.io/badge/NetBeans-29-blue)
![GitHub](https://img.shields.io/badge/GitHub-Repository-black)

FitTrack merupakan aplikasi Sistem Manajemen Fitness Center yang dikembangkan menggunakan Java Desktop Application dengan NetBeans IDE 29. Aplikasi ini dirancang untuk membantu pengelolaan data dan aktivitas pada pusat kebugaran secara lebih terstruktur, efisien, dan mudah digunakan.

## 📋 Deskripsi Proyek

Sistem ini dibuat sebagai proyek pengembangan perangkat lunak untuk mengelola berbagai kebutuhan operasional Fitness Center, seperti pengelolaan data member, jadwal latihan, serta informasi terkait aktivitas kebugaran.

Dengan adanya FitTrack, proses administrasi yang sebelumnya dilakukan secara manual dapat dilakukan secara digital sehingga meningkatkan efektivitas dan akurasi pengelolaan data.

---

## 🎯 Tujuan Proyek

* Membuat sistem manajemen Fitness Center berbasis desktop.
* Mempermudah pengelolaan data member dan aktivitas fitness.
* Mengurangi kesalahan pencatatan data secara manual.
* Menyediakan sistem yang mudah digunakan oleh administrator Fitness Center.

## 🚀 Cara Menjalankan Proyek

### 1. Clone Repository

```bash
git clone https://github.com/SakaZuki113/FitTrack.git
```

### 2. Buka di NetBeans

* Jalankan NetBeans IDE 29.
* Pilih **File → Open Project**.
* Pilih folder **FitTrack**.

### 3. Jalankan Program

Klik tombol **Run Project** atau tekan:

```text
F6
```
